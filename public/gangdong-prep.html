👥 §2. 보조 스탭 3인 역할 안내

각 보조 스탭에게 해당 섹션만 출력·배포 가능합니다.

📌 보조 1 — 접수·스티커·패치 담당

위치 입구 접수대

준비물 (접수대 비치)

  • 옷깃 스티커 5색 (각 40~60개)
  • 알쓰패치, 노담패치 (보냉가방) · 리플렛 2종
  • 알러지 안내문 (탁상 POP A3 — 시인성 강화)

작업 순서 단순화 v3

단계동작
1참가자 도착 → 인사 + 알러지 안내문(A3 POP) 안내 — "혹시 밴드·반창고 알러지 있으세요?" 구두 확인
2"평소 술 드시면 얼굴 빨개지시나요? 담배 피우세요?" 간단 확인 → 옷깃 스티커 1~2개 부착
3알쓰패치 1매 + 노담패치 1매 (흡연자 및 가족, 가족이 비흡연자라도 전달) + 리플렛 1부 지급
4"갤러리 쪽으로 안내드릴게요" 체험존 입장 유도
5혼잡 시 패치 부착 후 체험존 입장 유도 (보조3 신호에 따름)
💡 v3 단순화: 접수 설문지 작성 단계 삭제. 점심시간 빠른 흐름을 위해 구두 확인 + 스티커만으로 충분합니다.

스티커 색상 기준

색설문 답변
노랑Q1 "평소 빨개지지 않아요"
빨강Q1 "현재 빨개져요"
주황Q1 "예전엔 빨개졌는데 지금은 괜찮아요"
파랑Q2 흡연 "예"
초록Q3 가족 흡연 "예"
🎯 색상 의미는 참가자에게 공개하지 않습니다. 강사(APS 프로)가 시각 식별용으로만 사용.
붙이기 거부하시는 분 → "편하신 대로 괜찮습니다" 자연스럽게 넘어가세요.

혼잡 시: 대기 길면 "곧 10명 모이면 함께 부착 도와드릴게요" 안내 (AUDIT-K 먼저 작성 유도)

📌 보조 2 — 착석존·영상·만족도 담당

위치 착석 24석 테이블

준비물

  • 만족도 설문지 30부 (QR 미접속자 백업용 소량 — 메인은 QR 온라인 수집)
  • QR 코드 (탁상용)
  • 펜 30개

작업 순서 단순화 v3

단계동작
11차 판정 후 착석 안내
2테이블 착석 후 리플렛 뒷면 AUDIT-K 작성 및 가계도 작성 도움
3만족도 설문 QR 안내 — "휴대폰으로 QR 찍으시면 1분이면 됩니다"
4QR 접속 어려우신 분에게만 종이 설문지 배부 (30부 백업)
5종이 설문 작성 완료 시 회수
💡 워크시트 미사용 사유: L1 캠페인(90분 90명 압축 운영)은 체험·즉석 인지·만족도가 본질. 절주 서약 워크시트는 L2~L3 매트릭스(중독센터·보건소 단기개입)에서 본격 사용하므로 본 캠페인에서는 부담 없이 생략합니다.

스크린 영상

  • 무음 영상 2종 반복 재생 중인지 상시 확인
  • 멈추면 조용히 재시작

중요 멘트

  • "답안지 수거 안 해요. 가져가셔서 참고하시면 됩니다"
  • "점수 높다고 걱정 마세요. 오늘 참여가 시작이에요"
💡 점수 확인 및 안내는 매니저 담당. 보조 2는 착석 후 자리 관리·만족도·영상만.

📌 보조 3 — 동선·부착 도움·이탈자 담당

위치 공간 전체 순회 · 접수대 ↔ 체험존 이동 · 출구 근처

준비물

  • 리플렛 예비분

작업 순서 단순화 v3

단계동작
1전체 동선 관찰 — 접수 → 부착 → 갤러리 → 착석 → 출구 흐름 원활한지 확인
2집단 부착 시 APS 프로 보조 — 10명 단위 부착 지도할 때 옆에서 도움
3부착 후 온도센서 변색 확인 (자연스러운 손동작으로 보호필름 제거 도움)
4대기 길어지면 보조 1에 알림
5출구에서 만족도 설문(QR 또는 종이) 수거·전달 확인
6중도 이탈자 빠른 트랙: "시간 없어요" → 리플렛 전달 후 출구 안내
7피크 시간(12:30~13:00) 접수대·부착존 혼잡 시 복도에서 대기줄 관리 (대기 중 AUDIT-K 작성 안내)
8복도 대기 인원 20명 이상 시 보조1에게 알리고 접수대로 안내

부착 검증 절차 Pro Manual 기준

  1. 참가자 부착 완료 확인
  2. 온도센서 색 변화 관찰 (체온 전달되면 변색 → 투명에 가까워짐)
  3. 변색 시 보호필름 자연스럽게 제거 도움
  4. 7분 타이머는 APS 프로가 애플워치 음성 명령으로 관리 (보조 3은 혼잡시 시간 확인 도움)
  5. 따가움·화끈거리는 느낌 호소 → 즉시 매니저(간호사)에게 이관 (위험체질 의심 + 습식 찜질·이온음료 응대)

중요 멘트

  • "시간 급하시면 리플렛만 받아가셔도 괜찮아요"
  • "만족도 설문은 QR로 1분이면 됩니다"
  • "센서가 사라지는 게 보이죠? 측정에 적합한 피부온도입니다" (부착 직후·온도센서 변색 시점)
  • "따갑거나 화끈거리는 느낌이 있으신 분은 저에게 말씀해 주세요" (위험체질 조기 식별)
